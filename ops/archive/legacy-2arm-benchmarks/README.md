# Legacy 2-Arm Benchmarks Archive

This archive contains benchmark assets that compare only two arms, usually
`supervisor-only` versus `harness-run`.

These assets are no longer part of the active PNH benchmark surface because the
current benchmark acceptance contract requires every benchmark run to compare
all four operation modes:

- `supervisor-only`
- `supervisor-central`
- `normal-harness`
- `strict-harness`

Do not use these archived records as current harness decision evidence unless
the report explicitly states that it is reviewing historical 2-arm data.
